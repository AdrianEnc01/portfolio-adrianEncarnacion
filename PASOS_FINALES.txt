🐛 SOLUCIÓN DE PROBLEMAS

❌ "Git no es reconocido"
   → Instala Git: https://git-scm.com

❌ "Authentication failed"
   → Usa un token de acceso personal, no tu contraseña
   → Genera uno en: https://github.com/settings/tokens

❌ "remote origin already exists"
   → Ejecuta: git remote remove origin
   → Luego: git remote add origin TU-URL

❌ "failed to push some refs"
   → Ejecuta: git push -u origin main --force
